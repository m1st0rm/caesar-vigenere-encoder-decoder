# Caesar and Vigenere Encoder/Decoder App

## Description

This repo contains a desktop application developed in Python using Tkinter. The application provides the ability to encrypt and decrypt text data from a file using [Caesar](https://www.sciencedirect.com/topics/computer-science/caesar-cipher#:~:text=The%20Caesar%20cipher%20is%20a,another%20in%20a%20consistent%20fashion.) and [Vigen√®re](https://www.britannica.com/topic/Vigenere-cipher) encryption algorithms. Input of encryption and decryption keys is done by the user. The application processes only alphabetic characters of the Latin alphabet.

**Programming Language:** Python 3.12  
**Framework:** Tkinter
